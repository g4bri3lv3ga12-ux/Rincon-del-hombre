## Preguntas Frecuentes

### ¿Puedo vender productos?
**R:** Sí. El sistema está listo. Solo falta integrar pasarela de pagos (Stripe/PayPal) - es una mejora opcional.

### ¿Cómo subo productos?
**R:** Dos formas:
1. Panel Admin (más fácil) → Click "Nuevo Producto" → Llena formulario
2. Base de datos visual (Prisma Studio)

### ¿Es seguro?
**R:** Sí. Usa:
- JWT authentication (tokens seguros)
- bcrypt (encriptación de contraseñas)
- HTTPS (protocolo seguro)
- Rutas protegidas

### ¿Qué pasa si caída?
**R:** Vercel tiene 99.9% uptime. Además:
- Backups automáticos de BD
- Rollback automático si hay problema

### ¿Puedo cambiar colores?
**R:** Sí, muy fácil. Archivo `tailwind.config.ts`

### ¿Es lento?
**R:** No. Carga en <1 segundo gracias a Vercel CDN.

### ¿Otros dominios?
**R:** Sí. En Vercel puedes agregar múltiples dominios.

### ¿Cuánto cuesta alojar?
**R:** Vercel gratuito hasta cierto punto. Luego desde $20/mes.
